# 00 · Índice — Base de conocimiento de marca (Blanca Coutiño · Ceremonias Holísticas)

> **Qué es esto:** la base documental que recoge TODO el contexto de marca, negocio y contenido del proyecto. ## Cómo usar esta carpeta

1. Empieza por `CLAUDE.md` (raíz) para el panorama general y las reglas innegociables.
2. Según la tarea, abre el doc correspondiente:

| Si vas a… | Lee |
|---|---|
| Entender quién es Blanca y su autoridad | `01-identidad-marca.md` |
| Escribir o revisar copy / definir tono | `02-posicionamiento-voz.md` |
| Pensar audiencia, mercado, SEO, alianzas | `03-publico-mercado.md` |
| Trabajar las ceremonias (servicios) y su naming | `04-ceremonias.md` |
| Generar contenido de blog / educar sobre chakras | `05-marco-chakras.md` |
| Tratar fotos y activos visuales | `06-activos-fotos.md` |
| Construir el MVP (plan técnico) | `07-roadmap-mvp.md` |
| Arrancar la sesión de desarrollo | `../HANDOFF.md` |

## Convención de mantenimiento

- **Ground truth jerárquico:** si algo contradice a `CLAUDE.md`, gana `CLAUDE.md`. Los docs de `/docs` lo expanden, no lo contradicen.
- Al añadir información nueva del cliente: métela en el doc temático que corresponda y, si cambia una regla, refleja el cambio también en `CLAUDE.md`.
- Cada doc abre con una línea de **propósito**.
